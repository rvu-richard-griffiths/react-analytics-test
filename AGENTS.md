# UI Component Library

## Product

- Component library for react UI that are consumed in other applications.

### Components Required

- Button click
- Date picker
- Text box
- Pop up

### Capabilities

- Must provide context aware events for analytics that can integrate into any platform.
- Must be reproducible as part of the component library so events can be checked in isolation and as part of development.

## Tech Stack

- React
- Storybook

## Codebase Expectations

- Must contain README for service consumers that contains a quickstart.
- Must contain CONTRIBUTING for guidance on making changes.
- Must leverage TaskFile to provide consistency for codebase interactions.
- Must leverage conventional commits.
- Must contain .editorconfig and .gitattributes.
